# TestSessionGerome

Mini jeu **casse-brique** inspiré de l'esthétique Blackberry, en HTML/CSS/JavaScript vanilla.

## Lancer

Ouvre `index.html` dans ton navigateur.

## Contrôles

- `←` et `→` : déplacer la raquette
- `Entrée` : lancer la balle / démarrer
- Boutons à l'écran disponibles aussi pour mobile
